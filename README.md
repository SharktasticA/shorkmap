# SHORKMAP

**Notice 2026-06-17:** SHORKMAP's functionality has been merged into [SHORKSET](https://github.com/SharktasticA/shorkset) to unify settings-style SHORK Utilities into one. This repo will now be archived.

-----

A utility for SHORK OSes that changes the system's keyboard layout (keymap). It allows the user to choose one of the available keymaps (`/usr/share/keymaps/*.kmap.bin`) to set as the current keymap and is saved for subsequent reboots. It takes one argument (a keymap name); running it without an argument shows a list of possible keymaps.
